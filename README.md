# used-car-price-prediction
Machine Learning project for predicting used car prices using Linear Regression and Streamlit web application.


# 🚗 Used Car Price Prediction

This project is a Machine Learning web application that predicts the resale price of used cars based on different features such as:

- Car Name
- Company
- Car Age
- Kilometers Driven
- Fuel Type

## 🔧 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 📊 Machine Learning Model

- Linear Regression

## 🚀 Features

- User-friendly Streamlit interface
- Real-time car price prediction
- Interactive dropdowns and sliders
- Clean and responsive UI

## ▶️ Run Locally

```bash
streamlit run app.py
